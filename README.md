# RestauranteCriollo

Sistema de gestión de restaurante desarrollado como proyecto institucional para el curso **EFSRT III (Experiencias Formativas en Situaciones Reales de Trabajo III)**.

## 🚀 Tecnologías utilizadas
- Java 17
- Spring
- Spring Security
- Thymeleaf
- MySQL

## 📂 Funcionalidades principales
- Gestión de usuarios y roles
- Administración de mesas
- Registro y control de pedidos
- Gestión de platos y categorías
- Autenticación y autorización con Spring Security

## ⚙️ Configuración
El proyecto incluye un archivo de ejemplo para la configuración de la base de datos:
Para ejecutar la aplicación:
1. Copiar el archivo `application.properties.example` y renombrarlo como `application.properties`.
2. Editar las credenciales de conexión a la base de datos según tu entorno local.
3. Ejecutar el proyecto con Maven o desde tu IDE.
